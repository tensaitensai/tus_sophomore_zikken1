    \item $[$・・・記述内容が真(0)か偽(1)か評価する。引数の最後は必ず$]$でなければならない。

    1

    0

    1

    $ echo $?
    0

    $ echo $?
    0
    $ [ ! "abc" = "abe" ]

    0

                      =          & 等しければ真           \\ \hline

          \begin{lstlisting}
    $ cd jexpchi1
